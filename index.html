Intro

Super High Rate Server Information

LabyrinthRO's Super High Rate server was founded on 26 June 2020 bringing 1000/255 Elemental PvP to players that do not like to grind

The server is highly balanced having highly modified jobs to the point that even priests can fight!

Rates: 50000000x/50000000x/100%

High Rate Server Information

LabyrinthRO was founded on 23 June 2020 as a High Rate server bringing Race Systems and one of the largest costume databases to RO.

The High Rate server is largely still work in progress and we wish to grow the features according to players desires. So join us today in making a difference on the server.

Rates: 500x/500x/7%. Equip 6%, mvp equip 5%, boss equip 1%. Normal cards 5%, mvp cards 3%, boss cards 1%

Registration

To register an account, use an ID with more than 4 characters followed by _M or _F (e.g. account_M) to create the account. After registering for the first time, remove the _M or _F the second time you login.
